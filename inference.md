## **Inference of Commands**

## A.Directory Creation Commands

1.mkdir shell-assignment


Creates a new directory named shell-assignment.

2.cd shell-assignment


Changes the current working directory to shell-assignment.

## B. Commands Used Inside the Shell Script

3.#!/bin/bash


Specifies that the script should be executed using the Bash shell.

4.echo


Displays the specified text on the terminal.

5.$(date)


Retrieves the current system date and time.

## C. Script Execution Commands

6.chmod +x date\_time.sh


Grants execute permission to the shell script.

7../date\_time.sh


Executes the shell script from the current directory.

## D. Git Commands

8.git init


Initializes a new Git repository in the directory.

9.git add


Adds files to the Git staging area.

10.git commit


Saves the staged changes to the local Git repository.

11.git push


Uploads the committed files to the remote GitHub repository.
